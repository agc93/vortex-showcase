Alpha 1.4 Release

The new alpha release of Vortex Showcase
Saturday, October 24, 2020

This release is now available on Nexus Mods.

This release is a small feature update that adds new a new user setting to control your showcases a little better. If you haven't already, make sure you check out the notes for 0.1.1 and 0.1.0.

This version adds a new setting to control the ordering of mods in generated showcases. When you initially create a showcase (regardless of the game or format), your currently enabled mods will get loaded and you can now control the order those mods are read in. If you navigate to Settings > Interface and check the new Default Showcase sort order setting, you can set your mods to be sorted by their name (Alphabetical), or by Installation Time or Deployment Order.

This version requires Vortex 1.3+, older Vortex versions can continue to use 0.0.4. Extension authors might want to check the developer docs for detail on the new extensions API.

Get started with your own mod showcases using the action buttons in your Mods page, and make sure to report any issues you find or give any feedback.

Thanks to @ronamit on GitHub for this feature request!
